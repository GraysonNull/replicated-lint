
## `pod-no-latest`

Don't use `latest` for container versions





#### Examples:

*Incorrect*: Pod uses `latest` for container version

```yaml---
apiVersion: v1
Kind: Pod
metadata:
  name: my-pod
spec:
  containers:
    - name: redis
      image: redis:latest

```



*Correct*: Pod uses a specific version

```yaml---
apiVersion: v1
Kind: Pod
metadata:
  name: my-pod
spec:
  containers:
    - name: redis
      image: redis:3.2

```
